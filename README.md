#ASCII_Water

##How to build
In your console, type
```gcc ascii_water.c -o ascii_water```
and press Enter.

##Usage
In your console, type
```./ascii_water width height```
and press Enter.

##Warning
Does not look like water. The title sucks, lol.
Will not work on not UNIX-based systems.
