# ASCII_Water

[See a preview here](https://marimilch.de/projects#ascii-water)

## How to build
In your console, type
```sh build.sh```
and press Enter.

## Usage
In your console, type
```./ascii_water <width> <height>```
replacing parameters with desired values and press Enter.

## Warning
Will not work on not UNIX-based systems.
